## Features

1. Next Match.
2. Last Match.
3. Detail next match and last match.
4. Add to google calendar for next match.
5. List Team.
6. Detail Team.
7. Favorite match and team.

## Tech Stack
- [x] Kotlin
- [x] SQLite
- [x] Retrofit

API from: <a href="https://www.thesportsdb.com/" title="TheSportsDB">TheSportsDB</a>
